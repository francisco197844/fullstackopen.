ejemplo-de-envio-de-repositorio
Puede enviar todos los ejercicios de este curso al mismo repositorio o utilizar varios repositorios. Si envía ejercicios de distintas partes al mismo repositorio, utilice un esquema de nombres adecuado para los directorios.

Una estructura de archivos muy funcional para el repositorio de envíos es la siguiente:

parte0
Parte 1
  información del curso
  Unicafé
  Anécdotas
Parte 2
  directorio telefónico
  países
Coloque todo el repositorio de reacciones del proyecto en cada carpeta excepto la carpeta node_modules
